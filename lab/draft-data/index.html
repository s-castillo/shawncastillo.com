Lab / Draft Data

Real Fake Data Generator

Needed a tool that didn't exist at the right price point. Built it. Learned how to work with AI in the process.
Type Figma Plugin Proof of concept
Stack
Figma Plugin API
AI integration
The problem
Populating Figma tables with realistic fake data is tedious. Free plugins were too basic. Good ones were paywalled. So I built exactly what I needed.
What it does
A Figma plugin that rapidly populates table components with AI-generated data. Focused on fine-tune filters and settings that existing tools skipped.
What I learned
Building this showed me that AI can be a powerful tool to get something quickly made and functioning — a practical introduction to integrating AI into my design workflow.
Status
Local plugin — passing proof of concept. Publishing pending final tweaks.
